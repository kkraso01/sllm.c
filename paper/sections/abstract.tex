\begin{abstract}
We report an aligned evaluation pass for the finalized Adaptive Learning Component (ALC), a structured in-model memory module that updates state during inference without changing transformer weights. The central benchmark in this pass is SessionKV-DR (Session Key-Value Delayed Recall), designed to directly test whether ALC can write session-specific facts, retain them under delay/interference, retrieve them later, overwrite stale values, and preserve them across save/reload. We evaluate baseline, ALC no-write, and full ALC with multi-seed aggregation (5 seeds). This framing keeps claims mechanism-level: improved delayed session-specific recall under inference-time updates supports adaptive in-model memory behavior in this setup, while competitive no-write outcomes on some slices indicate remaining write-policy/interference limitations. TinyQA is retained as supporting semi-realistic evidence rather than primary proof of the memory claim.
\end{abstract}

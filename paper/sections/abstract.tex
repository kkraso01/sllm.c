\begin{abstract}
We study the Adaptive Learning Component (ALC), a structured in-model memory module that updates internal slot/key state during inference without changing core transformer weights. ALC uses differentiable routing for readout, routing-weighted exponential moving average writes, and an explicit persistence contract for save/reload. On controlled synthetic adaptation tasks using the finalized C implementation, ALC improves delayed recall over a no-memory baseline (recall MSE: 0.631 vs 0.880; recall accuracy: 0.141 vs 0.016), remains numerically stable over long update runs (0 NaN events in 10k updates), and preserves behavior exactly under state round-trip (max absolute behavior difference 0). Interface training further reduces retrieval loss (0.094 to 0.076), while adding moderate inference overhead (1.14$\times$ in our tiny CPU setup). We position ALC as a controllable mechanism for inference-time adaptation and discuss limits relative to retrieval-augmented and long-context systems.
\end{abstract}

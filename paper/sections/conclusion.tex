\section{Conclusion}
This pre-submission strengthening pass improves the ALC paper package with multi-seed robustness reporting, an added language-shaped delayed-recall benchmark, stronger baseline comparisons, and tighter claim discipline. The current evidence supports ALC as a controllable mechanism for inference-time adaptation with stable state dynamics and exact persistence semantics in this implementation.

We do \emph{not} claim that ALC solves lifelong memory or broadly outperforms retrieval/long-context systems. A stronger main-track version would require broader external benchmarks, stronger external baselines, and deeper analysis of write-policy robustness.

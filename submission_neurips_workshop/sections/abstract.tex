\begin{abstract}
This paper evaluates ALC, a structured in-model memory mechanism that performs runtime state updates during inference without modifying backbone weights. We focus on controlled empirical evidence using SessionKV-DR, which tests delayed recall, overwrite behavior, persistence, and interference sensitivity under a common baseline/no-write/full comparison protocol. Multi-seed aggregation shows that full ALC improves delayed session-specific recall in the targeted setting, while mixed no-write outcomes highlight remaining policy trade-offs. TinyQA is included as supporting evidence only. The contribution is a systems-mechanism study with careful claims and reproducible artifacts.
\end{abstract}

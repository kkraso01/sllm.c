\section{Introduction}
Mechanisms for adapting model behavior at inference time are attractive when post-deployment retraining is impractical. ALC provides structured mutable state with explicit read/write operations and persistence controls.

This NeurIPS workshop version emphasizes controlled empirical characterization: we benchmark delayed recall, overwrite, persistence, and interference under a consistent variant protocol.

The resulting evidence supports stable inference-time adaptation in the evaluated setup, while clearly separating mechanism-level findings from broader application claims.

\section{Introduction}
Inference-time adaptation remains a central challenge when newly observed information must be incorporated without frequent retraining. ALC addresses this through structured mutable state, leaving base parameters fixed while enabling explicit runtime writes and reads.

For a main-track audience, we treat evidence scope as a first-class issue. The current evaluation is designed to answer a mechanism question directly: whether structured inference-time writes improve delayed recall and update behavior under controlled interference.

Accordingly, SessionKV-DR is the primary benchmark, with TinyQA retained as supporting semi-realistic context. We report positive and mixed outcomes together to keep claims calibrated.

# submission_iclr_workshop

This folder is a venue-specific paper package derived from the canonical `paper/` materials in this repository.

## Central evidence used

- **Primary evidence:** SessionKV-DR delayed recall, overwrite, persistence, and capacity/interference tables (`tables/sessionkv_main.tex`, `tables/sessionkv_overwrite.tex`, `tables/sessionkv_capacity.tex`) sourced from artifacts under `results/`.
- **Supporting evidence:** TinyQA delayed QA table (`tables/tinyqa_results.tex`) sourced from `results/tinyqa_results.csv` and `results/tinyqa_summary.json`.
- **Comparative protocol:** baseline vs ALC no-write vs full ALC, with multi-seed summaries where available.

## Artifact provenance

All reported numbers in this package come from existing repository outputs generated by:

```bash
./scripts/run_alc_publication_suite.sh
```

The package copies canonical artifacts into local `results/` and `tables/` directories for self-contained inspection.

## Evidence limitations

- Evidence is mechanism-level and controlled, centered on SessionKV-DR.
- TinyQA is supportive and semi-realistic, not broad benchmark coverage.
- Results should not be interpreted as claims of lifelong-learning resolution, universal long-context superiority, or broad SOTA NLP gains.

\section{Introduction}
Architectural mechanisms for inference-time adaptation are attractive when systems need bounded, inspectable updates without parameter retraining. ALC provides such a mechanism through explicit mutable state and controlled routing/write operations.

This NeurIPS main-track package is intentionally conservative: evidence is presented as controlled mechanism validation, not broad downstream dominance. SessionKV-DR is central because it tests the exact runtime behaviors ALC is intended to provide.

We include TinyQA as secondary context and highlight both strengths and unresolved regimes to keep interpretation aligned with the data.

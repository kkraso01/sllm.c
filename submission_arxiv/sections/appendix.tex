\subsection{Build command}
To regenerate canonical artifacts in the repository root:
\begin{verbatim}
./scripts/run_alc_publication_suite.sh
\end{verbatim}

\subsection{Implementation detail notes}
ALC writes are routed using the same differentiable routing family used at read time, with runtime guards to bound unstable updates. Sidecar persistence serializes slot keys/values and validates shape compatibility at reload.

\subsection{Artifact traceability}
Primary tables in this submission are generated from the same canonical files under \texttt{paper/results/} and \texttt{paper/tables/}. This package intentionally reuses those artifacts to keep numbers identical across venue variants.

\begin{abstract}
This paper presents the Adaptive Learning Component (ALC), a structured in-model memory mechanism that updates runtime state during inference without changing base transformer weights. We evaluate ALC with a claim-aligned benchmark suite centered on SessionKV-DR (Session Key-Value Delayed Recall), which directly tests writing session-specific facts, retaining them under delay and interference, retrieving them later, overwriting stale values, and preserving state across save/reload. Across baseline, ALC no-write, and full ALC variants with multi-seed aggregation, results support mechanism-level gains in delayed session-specific recall and stable update behavior in this implementation. TinyQA is included as supporting semi-realistic evidence rather than primary proof of the memory claim. We report limitations explicitly: evidence is controlled and not a substitute for broad downstream benchmarking. The package emphasizes reproducibility, artifact traceability, and conservative interpretation.
\end{abstract}
